# 七音造声所

一个只用 Do、Re、Mi、Fa、Sol、La、Si 七个音符创作旋律的轻量网页音频生成器。

## 使用

直接打开 `index.html` 即可。也可以在当前目录启动任意静态文件服务器后访问。

- 点击七个彩色琴键，或使用键盘 `A S D F G H J` 演奏。
- 在 16 步节拍格中点亮音符，设置速度后循环播放。
- 可切换原声钢琴、电钢琴、正弦、三角、方波、锯齿波音色和三个八度。
- 点击“导出 WAV”保存当前旋律。

所有声音都由浏览器 Web Audio API 实时生成，不依赖音频素材。原声钢琴采用多层泛音与琴槌瞬态模拟，电钢琴采用 FM 合成；它们是轻量化合成音色，并非真实乐器录音采样。

发布到 HTTPS 网站后，可在手机浏览器中选择“添加到主屏幕”。首次完整打开后，应用也能离线使用。
